# Resonz v1 Android Starter

This is a starter Android project skeleton for **Resonz v1**.

Included:
- Compose-based UI shell
- locked design tokens
- main screen with preset buttons and hero Beat slider
- grouped Drift / Time / Tone controls
- Watch Sync row
- navigation skeleton
- core models and mapping helpers
- stub audio engine and watch-sync plumbing

Use this starter as the source tree an agent should extend, not redesign.
