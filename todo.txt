# 3. New audio files to add

Create these in `app/src/main/java/com/resonz/app/audio/`

## `SmoothedValue.kt`

One-pole smoother for beat, carrier, gains, etc.

```kotlin
package com.resonz.app.audio

class SmoothedValue(
    initial: Float,
    private val smoothingCoeff: Float
) {
    var current: Float = initial
        private set
    var target: Float = initial

    fun setTarget(value: Float) {
        target = value
    }

    fun reset(value: Float) {
        current = value
        target = value
    }

    fun next(): Float {
        current += (target - current) * smoothingCoeff
        return current
    }
}
```

Use different smoothers for:

* beat
* carrier
* body mix
* ocean mix
* master gain

## `BrownNoiseGenerator.kt`

```kotlin
package com.resonz.app.audio

import kotlin.random.Random

class BrownNoiseGenerator(seed: Int) {
    private val random = Random(seed)
    private var state = 0f

    fun next(): Float {
        val white = random.nextFloat() * 2f - 1f
        state = (state + 0.02f * white).coerceIn(-1f, 1f)
        return state * 3.5f
    }
}
```

## `OnePoleLowPass.kt`

```kotlin
package com.resonz.app.audio

import kotlin.math.PI
import kotlin.math.exp

class OnePoleLowPass(sampleRate: Float, cutoffHz: Float) {
    private var a = 0f
    private var z = 0f

    init { setCutoff(sampleRate, cutoffHz) }

    fun setCutoff(sampleRate: Float, cutoffHz: Float) {
        a = (1f - exp((-2.0 * PI * cutoffHz / sampleRate).toFloat()))
    }

    fun process(x: Float): Float {
        z += a * (x - z)
        return z
    }
}
```

## `OnePoleHighPass.kt`

```kotlin
package com.resonz.app.audio

class OnePoleHighPass {
    private var prevX = 0f
    private var prevY = 0f
    private var alpha = 0.995f

    fun setAlpha(value: Float) {
        alpha = value
    }

    fun process(x: Float): Float {
        val y = alpha * (prevY + x - prevX)
        prevX = x
        prevY = y
        return y
    }
}
```

## `DcBlocker.kt`

```kotlin
package com.resonz.app.audio

class DcBlocker {
    private var prevX = 0f
    private var prevY = 0f
    private val r = 0.995f

    fun process(x: Float): Float {
        val y = x - prevX + r * prevY
        prevX = x
        prevY = y
        return y
    }
}
```

## `BiquadBandPass.kt`

Use a simple RBJ biquad band-pass. This is the only mathy one, but worth it. Don’t skip it.

Fields:

This gives the ocean layer its character.

## `SoftClipper.kt`

```kotlin
package com.resonz.app.audio

import kotlin.math.tanh

object SoftClipper {
    fun process(x: Float): Float {
        val y = tanh(1.15f * x)
        val norm = tanh(1.15f)
        return (y / norm).coerceIn(-0.95f, 0.95f)
    }

    private fun tanh(x: Float): Float = tanh(x.toDouble()).toFloat()
}
```

## `SineOscillator.kt`

```kotlin
package com.resonz.app.audio

import kotlin.math.PI
import kotlin.math.sin

class SineOscillator {
    private var phase = 0.0

    fun next(freqHz: Float, sampleRate: Float): Float {
        val out = sin(phase * 2.0 * PI).toFloat()
        phase += freqHz / sampleRate
        if (phase >= 1.0) phase -= 1.0
        return out
    }
}
```

# 5. Rebuild `RealTimeAudioEngine.kt`

## New responsibilities

`RealTimeAudioEngine` should do only this:

* own AudioTrack
* own oscillators/noise/filter render objects
* own smoothed runtime parameters
* stream audio continuously while active
* expose transport methods

## Internal state

Replace your current state mess with:

```kotlin
enum class EngineState {
    IDLE, STARTING, RUNNING, PAUSING, PAUSED, STOPPING, STOPPED
}
```

## Important transport rule

The render loop must continue while:

* starting fade-in
* pausing fade-out
* paused at zero gain
* stopping fade-out

Do **not** use:

```kotlin
while (isActive && state == EngineState.PLAYING)
```

Use something like:

```kotlin
while (isActive && state != EngineState.STOPPED && state != EngineState.IDLE)
```

Or better, keep a separate boolean `renderLoopActive`.

## Gain behavior

Use a smoothed master gain:

* fade in to `0.26f`
* fade out to `0f`

When paused:

* keep track alive
* keep rendering silence or near-silence
* do not release track

When stopped:

* fade to zero
* then stop/release track
* then set state to `STOPPED`

# 6. Core render formula

This is the actual sound recipe.

For every stereo frame:

## A. Binaural core

```kotlin
val leftFreq = carrier - beat * 0.5f
val rightFreq = carrier + beat * 0.5f
```

Then:

```kotlin
val leftCore = leftOsc.next(leftFreq, sampleRate)
val rightCore = rightOsc.next(rightFreq, sampleRate)
```

For Focus only, add tiny harmonics:

```kotlin
val leftH2 = leftH2Osc.next(leftFreq * 2f, sampleRate) * harmonic2Mix
val rightH2 = rightH2Osc.next(rightFreq * 2f, sampleRate) * harmonic2Mix
val leftH3 = leftH3Osc.next(leftFreq * 3f, sampleRate) * harmonic3Mix
val rightH3 = rightH3Osc.next(rightFreq * 3f, sampleRate) * harmonic3Mix
```

Then:

```kotlin
val leftMain = (leftCore + leftH2 + leftH3) * 0.26f
val rightMain = (rightCore + rightH2 + rightH3) * 0.26f
```

## B. Body layer

Use a separate oscillator:

```kotlin
val bodyFreq = carrier * 0.75f
val body = bodyOsc.next(bodyFreq, sampleRate) * bodyMix
```

Then spread it slightly:

```kotlin
val bodyL = body * 0.98f
val bodyR = body * 1.02f
```

## C. Ocean layer

From `OceanLayerRenderer`:

```kotlin
val (oceanL, oceanR) = oceanRenderer.nextFrame(oceanMix)
```

## D. Final mix

```kotlin
var left = leftMain + bodyL + oceanL
var right = rightMain + bodyR + oceanR

left = dcBlockL.process(left)
right = dcBlockR.process(right)

left *= masterGain
right *= masterGain

left = SoftClipper.process(left)
right = SoftClipper.process(right)
```

Then write to buffer.

# 7. Ocean layer settings

Build the ocean with **filtered brown noise**, not WAV files.

## Suggested settings

* HPF: around `25–30 Hz`
* LPF: around `1400–1600 Hz`
* band-pass 1 center: `360 Hz`
* band-pass 2 center: `850 Hz`
* Q around `0.7–1.0`

## Modulation

Use very slow LFOs:

* `0.035 Hz`
* `0.07 Hz`
* `0.05 Hz` swell

Keep modulation subtle:

* ±12–15%

## Mix amount

* Deep Sleep: `0.045`
* Drift to Sleep: `0.05`
* Earth Sync: `0.04`
* Focus: `0.02`

That is enough. Don’t overcook it.
